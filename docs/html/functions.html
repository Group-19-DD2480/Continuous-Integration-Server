ci_server.build_project(path: str) → bool¶

Fetches all python files in the directory given by path and runs a compile check on it. Return true if the check succeeds.

Parameters:

path (str) – String representing the path to the directory to be compiled. An empty directory is considered a valid path and program (returns True)

Returns:

True if all python files within the path compile without errors. False if any file compile with an error, path is not directory or the path does not exist.

ci_server.update_github_status(url: str, state: str, github_token: str) → int¶

Updates the status of a commit on github.

This function sends a commit status update to github using the Statuses API. The status can be one of the following: “success”, “failure”, “pending”, or “error”.

Parameters:

  • url (str) – The github API endpoint for updating commit statuses.

  • state (str) – The status to be sent to github. Must be one of: “success”, “failure”, “pending”, or “error”.

  • github_token (str) – The authentication token used to access the github API.

Returns:

The HTTP status code returned by github.

Return type:

int
